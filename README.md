# Omicron-CEP-JavaFX

Programa simples para consultar CEPs, escrito em Java com JavaFX.

**Nota:** Programa desenvolvido apenas para estudos com Java.

## Informações

Desenvolvido com:

* Java
* JavaFX
* SQLite
* JSON

## Compilar

Compilar no Linux:

```
./gradlew assemble
```

**Nota:** Desenvolvido e testado somente no Linux. Não testado no Windows.
